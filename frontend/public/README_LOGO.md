# Logo faylini qo'yish

`logo.png` faylini `public/` papkasiga qo'ying.

**Tavsiya etilgan o'lchamlar:**
- Kichik logo: 50x50px (header uchun)
- Katta logo: 200x200px yoki 300x300px (login page uchun)

**Format:** PNG (shaffof fon bilan)

Agar logo fayli bo'lmasa, avtomatik SVG fallback ishlatiladi.
